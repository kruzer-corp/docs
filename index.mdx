O **iPaaS Kruzer** é um ambiente robusto e escalável para conectar, orquestrar e automatizar integrações entre sistemas internos e externos. Ele abstrai a complexidade da comunicação entre APIs, bancos de dados, filas e serviços, permitindo que engenheiros construam fluxos complexos com agilidade.

## Exemplo Prático

Imagine sincronizar dados de estoque entre um ERP e uma plataforma de e-commerce. Com o iPaaS Kruzer, você pode criar um fluxo que:

1. **Captura** atualizações de estoque do ERP via webhook
2. **Transforma** os dados para o formato esperado pela API do e-commerce
3. **Envia** as atualizações automaticamente, tratando erros e aplicando retries

```typescript
import { MySqlDataSource, RestDataSource, KrzLogger } from '@kruzer/idk';

const logger = new KrzLogger('info', 'sync-estoque');
const erp = new MySqlDataSource('erp-database');
const ecommerce = new RestDataSource('ecommerce-api');

export default async function run() {
  // Busca produtos atualizados no ERP
  const produtos = await erp.query(
    'SELECT sku, quantidade FROM estoque WHERE atualizado_em > :data',
    { data: { val: '2024-01-01' } }
  );

  // Envia para o e-commerce
  for (const produto of produtos) {
    await ecommerce.request('PUT', `/products/${produto.sku}/stock`, {
      body: { quantity: produto.quantidade }
    });
  }

  logger.info('Sincronização concluída', { total: produtos.length });
}
```
